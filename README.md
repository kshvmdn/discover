## explore-github

### Installation

##### Requirements:

  - Python 3+
  - Redis
  - Node.js (for development)

- Clone project.

  ```sh
  git clone https://git@github.com/kshvmdn/explore-github.git
  cd explore-github
  ```

- Install dependencies.

  ```sh
  pip install -r ./server/requirements.txt
  cd app && npm install && cd ..
  ```
